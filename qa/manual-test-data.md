# Manual Test Data Policy

이 저장소에는 실제 또는 예시 소비 내역 파일을 포함하지 않습니다.

수동 테스트가 필요하면 앱을 브라우저에서 열고, 테스트가 끝난 뒤 브라우저 저장 데이터를 초기화하세요. 테스트 중 내려받은 백업 JSON, 엑셀 파일, CSV 파일은 저장소 밖에 보관하고 커밋하지 않습니다.

## 수동 테스트 원칙

- 실제 카드 내역이나 은행 내역을 저장소에 넣지 않습니다.
- 테스트용 엑셀, 백업 JSON, 소비 기록 샘플을 커밋하지 않습니다.
- 화면 동작 확인은 브라우저의 임시 저장소에서만 수행합니다.
- 테스트 후 `설정/관리`의 선택 초기화 또는 브라우저 사이트 데이터 삭제로 테스트 데이터를 지웁니다.

## 배포 전 확인

```powershell
git status --short
```

위 명령 결과에 `.xlsx`, `.xls`, `.csv`, `.json`, `.ps1`, `.cmd` 같은 개인 데이터 또는 로컬 도구 파일이 포함되지 않아야 합니다.
