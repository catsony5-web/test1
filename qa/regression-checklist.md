# Regression Checklist

## 데이터 유지
1. 기존 localStorage 데이터가 삭제되지 않는다.
2. 기존 IndexedDB private data가 유지된다.
3. 기존 저장 키 이름이 바뀌지 않는다.
4. 자동 저장 스냅샷과 백업/복구가 동작한다.

## 가져오기와 분류
1. 엑셀 업로드가 정상 처리된다.
2. 중복 거래 건너뛰기가 동작한다.
3. 카드 결제와 이체 내역 파싱이 유지된다.
4. 자동 분류 규칙이 기존과 같은 결과를 낸다.
5. 미분류 해결과 규칙 추가가 동작한다.
6. 분류 규칙 수정, 삭제, 기본 복원이 동작한다.

## 화면별 기능
1. 대시보드 요약, 섹터 카드, TOP 상세 항목이 표시된다.
2. 상세 내역 검색, 필터, 정렬이 동작한다.
3. 소비 달력 월 이동, 날짜 클릭, 상세 리스트가 동작한다.
4. 월별 정리 표와 그래프가 표시되고 수입 클릭 이동이 동작한다.
5. 월별 요약 그래프, 섹터 선택, 월 선택이 동작한다.
6. 수입 단건 입력, 일괄 붙여넣기, 수정, 삭제가 동작한다.
7. 고정 지출 등록, 수정, 삭제, 반영이 동작한다.
8. 소모품 사용 기록, 필터, 차트가 동작한다.
9. 전체 내역 수동 입력이 동작한다.

## UI 유지
1. 기존 레이아웃과 디자인이 변경되지 않는다.
2. body 전체 가로 스크롤이 생기지 않는다.
3. 반응형 메뉴와 카드 그리드가 무너지지 않는다.
4. 브라우저 콘솔 오류가 없다.
