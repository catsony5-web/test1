# Smoke Checklist

1. 앱을 새로고침했을 때 콘솔 오류가 없는지 확인한다.
2. 대시보드가 첫 화면에서 정상 렌더링되는지 확인한다.
3. 왼쪽 메뉴 탭 전환이 모두 동작하는지 확인한다.
4. 카드/이체 엑셀 파일 선택 버튼이 열리는지 확인한다.
5. 월별 정리, 월별 요약, 소비 달력, 상세 내역이 비어 있지 않게 표시되는지 확인한다.
6. 수입 입력 탭에서 월 선택, 기록 목록, 돌아가기 버튼이 보이는지 확인한다.
7. 분류 규칙 탭에서 목록, 수정, 삭제 버튼이 보이는지 확인한다.
8. 새로고침 후 기존 저장 데이터가 유지되는지 확인한다.
