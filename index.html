대시보드

년도 지출정리

연도별 흐름 안에서 월별 결제액, 실지출, 수입, 잔액, 자산 변화를 비교합니다.

수입 - 실 지출 추이

월별 수입 대비 남은 금액과 부족 금액을 확인합니다.

수입 입력

급여, 환급, 지원금, 기타 수입을 날짜별로 기록합니다.

단건 입력

수입 한 건을 빠르게 추가합니다.

일괄 붙여넣기

엑셀에서 날짜 / 내용 / 금액 형식으로 여러 줄을 붙여넣습니다.

날짜는 YYYY.MM.DD 또는 YYYY-MM-DD, 금액은 숫자 또는 쉼표 포함 형식을 사용할 수 있습니다.

저장된 수입 기록

최근 수입부터 확인하고 필요한 항목은 바로 수정합니다.

고정 지출

매월 반복되는 카드값, 보험료, 월세, 통신비 같은 예정 지출을 미리 등록합니다.

고정 지출 등록

저장해도 실제 지출에는 합산하지 않고 예정 지출로 표시합니다.

고정 지출 일괄 입력

날짜 / 지출명 / 금액 형식으로 여러 줄을 붙여넣으면 매월 반복 지출로 등록합니다.

등록된 고정 지출

목록에서 수정, 삭제, 일시중지, 이번 달 실제 지출 반영을 할 수 있습니다.

월별 섹터 요약

월별 섹터별 소비 비중과 변화를 비교합니다.

섹터별 월별 증감

기간과 섹터를 선택해 전월 대비 소비가 얼마나 늘거나 줄었는지 확인합니다.

선택 월 섹터 비교

선택 월의 총지출, 가장 큰 섹터, 선택 섹터 비중과 미분류 금액을 함께 봅니다.

상세 섹터 비교

선택한 월의 섹터별 금액과 주요 세부항목 TOP 3를 비교합니다.

각 섹터 리포트

선택한 월과 섹터의 세부 소비 패턴을 더 자세히 분석합니다.

월별 섹터 매트릭스

기간 내 모든 월의 섹터별 금액을 표로 비교하는 보조 데이터입니다.

공모주

청약 일정, 배정 주수, 매도 금액을 기록하고 매도금액과 공모가 기준으로 손익을 자동 계산합니다.

공모주 달력

청약, 환불, 상장, 매도 일정을 월별 달력으로 확인합니다.

공모주 기록 입력

매도 금액만 입력해도 공모가 기준으로 손익을 계산합니다. 배정 주수는 기록용으로만 사용합니다.

자동 일정 후보

전체 공모주 기록

청약 상태, 증권사, 월, 종목명으로 기록을 찾고 매도 결과를 관리합니다.

상세 내역

필터 조건에 맞는 거래를 조회하고, 전체 보기에서 정산금과 할부 정보를 수정합니다.

카드 안의 + 내역 추가로 입력하고, 전체 보기에서 정산금을 수정할 수 있습니다.

과거 거래 입력

카드 내역 엑셀이 없는 과거 소비 기록을 붙여넣고, 선택한 분류로 한 번에 저장합니다.

과거 거래 일괄 입력

카드 내역이 없는 과거 소비 기록을 날짜 / 내용 / 금액 형식으로 붙여넣어 등록합니다. 월은 날짜에서 자동 추출됩니다.

전체 과거 거래 기록

일괄 입력으로 저장한 거래를 월, 섹터, 세부항목, 검색어 기준으로 확인하고 수정합니다.
